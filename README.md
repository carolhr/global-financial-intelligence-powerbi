## 📌 Sobre o Projeto
Este repositório contém uma central de comando de inteligência de mercado desenvolvida no Power BI Desktop. O projeto utiliza o conjunto de dados nativo de performance financeira global (`financials`) e se destaca por uma interface disruptiva em estilo *Glassmorphism / Dark Cyberpunk*, simulando o painel operacional de uma fintech ou terminal financeiro sênior.

## 🛠️ Tecnologias e Métodos Aplicados
* **Power BI Desktop:** Modelagem de dados, tratamento de eixos escuros, aplicação de visuais avançados e engenharia de navegação nativa.
* **Figma (UI/UX):** Prototipagem tridimensional do background, aplicando camadas translúcidas (60% a 75% de opacidade), *Background Blur* e efeitos de emissão de luz por reflexo (*Glow Effects* via *Drop Shadow* sem deslocamento espacial).

## 🗂️ Arquitetura das Páginas e Estrutura de UX
O projeto foi segmentado em 3 visões complementares acessadas por um **Navegador Vertical Interativo** na barra lateral esquerda:
1. **Visão Geral:** Cockpit macro com 4 KPIs consolidados de Faturamento, Descontos, Lucro Líquido e Unidades Vendidas, associado a um gráfico de dispersão (análise de mix de produtos lucrativos) e ondas suaves de sazonalidade temporal.
2. **Produtos:** Foco granular no portfólio de vendas, utilizando gráficos de funil neon e dispersão para identificar quais linhas geram faturamento alto mas possuem margens de lucro comprimidas.
3. **Geográfico:** Mapeamento de performance internacional cruzando os lucros em colunas eletrônicas com o ranking de descontos agressivos concedidos em cada nação.

## 🎨 Diferenciais de Interface (UI)
* **Contraste Controlado:** Substituição de todos os textos pretos nativos por fontes em branco e cinza claro, garantindo acessibilidade e leitura em ambientes de tema escuro (*Dark Mode*).
* **Consistência de Identidade:** Uso estrito de uma paleta de cores eletrônicas (Ciano `#00F2FE` e Roxo Neon) para categorizar e diferenciar blocos financeiros de blocos operacionais.
